크몽 종합소득세 신고 계산기

Step 1. 세금계산서 자동 다운로드 → Step 2. 파일 업로드 → 수익 현황 및 종합소득세 자동 계산

Step 1 — 크몽 세금계산서 자동 다운로드
아래 버튼을 북마크바로 드래그해 두고, 크몽 수익관리 페이지에서 클릭하면 전년도 전체 XLS 자동 다운로드
01
북마크바로 드래그
아래 파란 버튼을 브라우저 북마크바로 드래그 (안 보이면 Ctrl+Shift+B)
02
크몽 페이지 이동
크몽 로그인 후 수익관리 → 월별 수수료 세금계산서 페이지로 이동
→ 크몽 세금계산서 페이지 바로가기
03
북마크 클릭
저장한 북마크 클릭 → 전년도 전체 월 XLS 자동 다운로드
04
아래에 업로드
다운로드된 파일을 Step 2 업로드 영역에 드래그 또는 선택
⬇  크몽 세금계산서 전년도 다운로드 ↗  크몽 세금계산서 페이지 열기 💡 드래그 버튼은 클릭 말고 북마크바로 드래그하세요
Step 2 — 세금계산서 파일 업로드
Step 1에서 다운로드한 XLS 파일을 여기에 업로드하면 분석이 시작됩니다
